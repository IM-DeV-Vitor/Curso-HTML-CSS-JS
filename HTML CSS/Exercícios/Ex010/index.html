Lista de Definições

Liguagem HTML
HTML é a sigla para Hypertext Markup Language, que significa Linguagem de Marcação de Hipertexto. É uma linguagem de marcação de texto que define a estrutura e o significado do conteúdo de uma página web. O HTML é o componente básico da web, sendo utilizado para criar páginas, inserir imagens e vídeos, e definir elementos como parágrafos e títulos. O HTML é usado em: Desenvolvimento de páginas web, Desenvolvimento de aplicativos para dispositivos móveis, Desenvolvimento de jogos. O HTML é criado por meio de marcadores (tags) e atributos, que definem como o conteúdo deve ser apresentado no navegador web.
Linguagem CSS
CSS (Cascading Style Sheets) é uma linguagem de estilo que descreve como um documento é apresentado. Ela é usada para estilizar páginas web escritas em HTML ou XML. O CSS foi criado em 1996 pelo World Wide Web Consortium (W3C). A sua função é separar o conteúdo de uma página web da sua apresentação visual.
JavaScript
O JavaScript é amplamente categorizado como uma linguagem de criação de scripts ou uma linguagem interpretada. O código em JavaScript é interpretado, ou seja, é traduzido diretamente no código de linguagem de máquina subjacente por um mecanismo de JavaScript.
